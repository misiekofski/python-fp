# Functional programming in Python 3.11

### Simple exercised to refactor OOP code in proper FP way, tests included!

### How to install
`pip install -r requirements.txt` # you need pytest at least

### Structure of files
```
python-fp
│   README.md
│   requirements.txt    
│
└───before-fp
│   │   group.py
│   │   test_groups.py
│   │   user.py
|
└───after-fp
│   │   group.py 
│   │   test_groups.py
│   │   user.py
```
